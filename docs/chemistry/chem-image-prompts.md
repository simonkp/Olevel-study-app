# Image / infographic prompts (Nano Banana / any image model)

Use **flat vector / textbook diagram** style, **dark background** optional to match the app, **no watermark**, **clear labels**, **MOE O-Level Chemistry** level. Aspect ratio **16:9** or **4:3**.

## Theme 1 — Matter

| Topic | Prompt |
|-------|--------|
| **1 Experimental design** | Diagram of a fair test: beaker labelled “reactants”, thermometer, timer, one variable highlighted “changed”, others “kept same”. Arrows: independent → dependent. Clean vector, exam revision poster. |
| **2 Purification** | Side-by-side: simple distillation apparatus (flask, condenser, receiving flask); paper chromatography tank with baseline and rising solvent front. Labels only, no photorealism. |
| **3 Kinetic theory** | Three panels: solid (ordered grid of spheres), liquid (disordered close), gas (sparse fast arrows). Caption-style labels. Educational infographic. |
| **4 Atomic structure** | Bohr-style atom: nucleus with +p/n, shells 2,8,1 for sodium. Legend for proton, neutron, electron. |
| **5 Bonding** | Split panel: Na⁺ and Cl⁻ in crystal lattice (2D grid); H–H covalent with shared pair; metallic “sea of electrons” with + ions. |
